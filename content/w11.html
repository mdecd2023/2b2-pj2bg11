7段顯示器圖片檔案

記分板製作，一開始決定使用7段顯示器的方式去製作，並設計轉軸，但是過程並不順利，直到我發現，其他組的人利用顏色的改變來顯示數字，這十分的有趣，所以我參考她的程式並做出我自己的理解，和做出一點改變。

function sysCall_init()

bubbleRob = sim.getObject('/bubbleRob1')
ball = sim.getObject('/Sphere')
bubbleRob2 = sim.getObject('/bubbleRob2')
bubbleRob3 = sim.getObject('/bubbleRob3')

initialPosition = sim.getObjectPosition(bubbleRob, -1)
initialOrientation = sim.getObjectOrientation(bubbleRob, -1)

initialPosition2 = sim.getObjectPosition(bubbleRob2, -1)
initialOrientation2 = sim.getObjectOrientation(bubbleRob2, -1)

initialballPosition = sim.getObjectPosition(ball, -1)
initialballOrientation = sim.getObjectOrientation(ball, -1)

initialPosition3 = sim.getObjectPosition(bubbleRob3, -1)
initialOrientation3 = sim.getObjectOrientation(bubbleRob3, -1)

sensor = sim.getObject('./sensor')

score1 = 0

score3 = 0 

以上這些程式是在設定物體的對應，其中可以看到各個物體的對應和分數設置為0，再來很重要的是設定物體的對應位置，接下來在進球後要將所有物體重製。

s0={1,1,1,1,1,1,0}
s1={0,0,0,1,1,0,0}
s2={1,1,1,1,0,0,1}
s3={0,1,1,1,0,0,1}
s4={0,0,0,1,1,1,1}
s5={0,1,1,0,1,1,1}
s6={1,1,0,0,1,1,1}
s7={0,0,1,1,1,0,0}
s8={1,1,1,1,1,1,1}
s9={0,1,1,1,1,1,1}
s={s0,s1,s2,s3,s4,s5,s6,s7,s8,s9}

這是數字的儲存庫，對應著每一個7段顯示器中的一條，0和1就是不同的顏色，則s中的就是每一條的編號。

for j = 0,6,1 do
local t = sim.getObject('./a['..j..']')

if (s[1][j+1]==1) then
sim.setShapeColor(t, nil, sim.colorcomponent_ambient_diffuse, {0, 1, 0})
else
sim.setShapeColor(t, nil, sim.colorcomponent_ambient_diffuse, {1, 1, 1})
end
end

for迴圈的應用，sim.setShapeColor則是改變顏色後面的則是顏色編號，local是局部參數，sim.getObject('./a['..j..']')這是導入每一條的。

for j = 0,6,1 do
local d = sim.getObject('./b['..j..']')
if (s[1][j+1]==1) then
sim.setShapeColor(d, nil, sim.colorcomponent_ambient_diffuse, {0, 1, 0})
else
sim.setShapeColor(d, nil, sim.colorcomponent_ambient_diffuse, {1, 1, 1})
end
end

這則是十位數的數字。

function sysCall_actuation()
result = sim.readProximitySensor(sensor)
if (result > 0) then

sim.setObjectPosition(bubbleRob, -1, initialPosition)
sim.setObjectOrientation(bubbleRob, -1, initialOrientation)


sim.setObjectPosition(bubbleRob2, -1, initialPosition2)
sim.setObjectOrientation(bubbleRob2, -1, initialOrientation2)


sim.setObjectPosition(ball, -1, initialballPosition)
sim.setObjectOrientation(ball, -1, initialballOrientation)

sim.setObjectPosition(bubbleRob3, -1, initialPosition3)
sim.setObjectOrientation(bubbleRob3, -1, initialOrientation3)

這裡很重要，result = sim.readProximitySensor(sensor)是導入偵測器的資料，當資料大於0時，所有的物體都會回到一開始設定的位置。

if (score1 < 9) then
score1 = score1 + 1
os.execute("sleep 1")
for i = 0, 9, 1 do
if (score1 == i) then
for j = 0, 6, 1 do
local d = sim.getObject('./a['..j..']')
if (s[i+1][j+1] == 1) then
sim.setShapeColor(d, nil, sim.colorcomponent_ambient_diffuse, {0, 1, 0})
else
sim.setShapeColor(d, nil, sim.colorcomponent_ambient_diffuse, {1, 1, 1})
end
end
end
end

score2 = score1

這裡很好理解，就是當score1小於9時，則score1+1，並將score2和score1相同。

elseif (score3 < 9) then
score1 = 0
score3 = score3 + 1
os.execute("sleep 1")
for i = 0, 9, 1 do
if (score3 == i) then
for j = 0, 6, 1 do
local t = sim.getObject('./b['..j..']')
if (s[i+1][j+1] == 1) then
sim.setShapeColor(t, nil, sim.colorcomponent_ambient_diffuse, {0, 1, 0})
else
sim.setShapeColor(t, nil, sim.colorcomponent_ambient_diffuse, {1, 1, 1})
end
end
end
end
score4 = score3

這裡是控制十位數的。
